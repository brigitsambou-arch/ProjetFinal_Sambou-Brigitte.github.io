=========================
QUALITÉ — Résumé
=========================

1) Accessibilité

Problème :
Le site ne possédait pas encore de vérification complète de l'accessibilité.

Correction :
Ajout de balises HTML sémantiques (header, nav, main, section, footer), d'un seul H1, d'attributs alt pour les images, de labels pour les champs du formulaire et d'une navigation claire.

Résultat :
Le site est plus accessible et respecte les bonnes pratiques HTML.

-------------------------------------

2) Performance

Problème :
Certaines ressources pouvaient ralentir le chargement.

Correction :
Utilisation d'un seul fichier CSS, d'un seul fichier JavaScript, image de profil optimisée et chargement du script avec l'attribut defer.

Résultat :
Le site se charge rapidement avec un nombre réduit de ressources.

-------------------------------------

3) Corrections / Robustesse

Problème :
Le formulaire acceptait des champs vides et des adresses email incorrectes.

Correction :
Ajout d'une validation JavaScript des champs obligatoires et de l'adresse email, avec affichage de messages d'erreur ou de succès.

Résultat :
Le formulaire fonctionne correctement et guide l'utilisateur en cas d'erreur.
